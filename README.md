# React Google Calendar

## [Site link](https://calendar-vitalekp.netlify.app/)

## The tech stack is:

- React
- React Hooks
- JS (ES6)
- Sass (Scss)
- Webpack
- REST API

## Commands:

- `npm start` - to start project
- `npm run build` - to build production
- `npm run test` - tests with jest
- `npm run line` - check syntacks with eslint
